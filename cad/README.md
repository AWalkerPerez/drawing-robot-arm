# CAD

## Folders
- `fusion360/` — editable source CAD (Fusion 360)
- `print_files/3mf/` — print-ready 3MF exports (recommended)
- `print_files/stl/` — print-ready STL exports (most compatible)
- `renders/` — images used in the main README

## Notes
3MF files are the easiest way to reproduce the exact prints.
Fusion 360 files are provided for modification/customisation.
